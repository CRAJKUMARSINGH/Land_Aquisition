## üë§ For Claimants

### Step 1: Understand Your Case

#### 1.1 Start with Home Page
- View success statistics
- Understand key success factors
- Check if your case type is covered

#### 1.2 Read Legal Research
1. Click **"Legal Research"** in navigation
2. Browse available documents:
   - Executive Summary (start here)
   - Successful Cases Summary
   - Courts & Filing Guide
3. Use search to find specific topics

### Step 2: Analyze Similar Cases

#### 2.1 Browse Successful Cases
1. Click **"Case Management"**
2. Review 4 successful cases:
   - Kalabhai Hadabhai (Delay condoned)
   - State of MP vs Chhakkilal (Enhancement upheld)
   - Anand vs SLAO (Parity principle)
   - LAO vs Dodla Chinnaiah (Comparable evidence)

#### 2.2 Filter Cases
- **By State**: Select your state from dropdown
- **By Year**: Filter by year (2024-2025)
- **By Result**: View only successful cases
- **By Strategy**: Filter by legal strategy used

#### 2.3 Learn from Rejected Case
1. Click on **"Rejected Cases"** tab
2. Study Nirubhai Bhurabhai case
3. Understand what NOT to do:
   - Don't wait too long (772 days rejected)
   - Don't "fence-sit" (wait for others)
   - File within limitation period

### Step 3: Check Your Jurisdiction

#### 3.1 Find Your Court
1. Click **"Jurisdiction Guide"**
2. Select your state
3. View:
   - High Court name
   - Principal seat location
   - Circuit benches (if any)
   - Coverage area
   - Official website

#### 3.2 Determine Filing Location
- If your district is listed under a bench ‚Üí File at that bench
- If not listed ‚Üí File at principal seat
- Example: Dharwad district ‚Üí Dharwad Bench of Karnataka HC

### Step 4: Assess Your Risk

#### 4.1 Check Success Probability
1. Click **"Risk Analysis"**
2. Review success factors:
   - **Parity Principle**: 85-95% success
   - **Comparable Evidence**: 70-85% success
   - **Prompt Action**: Critical
   - **Under Protest**: Essential

#### 4.2 Identify Your Evidence Type
Match your evidence to success rates:
- Same notification judgment ‚Üí 85-95% success
- Same locality judgment ‚Üí 70-85% success
- Multiple sale deeds + guidelines ‚Üí 60-75% success
- Single sale deed ‚Üí 40-50% success
- No evidence ‚Üí Don't file (waste of money)

### Step 5: Prepare Your Documents

#### 5.1 Access Legal Templates
1. Click **"Document Drafting"**
2. Choose template type:
   - **Section 18**: Reference Application
   - **Section 54**: First Appeal to High Court

#### 5.2 Review Template Sections
- Click **"View Details"** to expand
- Read all sections carefully
- Note placeholders (marked with [brackets])
- Download or copy template

#### 5.3 Gather Required Documents
Based on template, collect:
- Land ownership documents
- Award/Judgment copy
- Comparable sale deeds (minimum 3)
- Collector's Guidelines
- Comparable judgments
- Photos of land
- Location map

### Step 6: Consult a Lawyer

#### 6.1 Prepare for Consultation
Take with you:
- All documents collected
- Printed templates from this app
- Notes on similar successful cases
- Your evidence assessment

#### 6.2 Questions to Ask
- What is my success probability?
- What are the costs (lawyer + court fees)?
- How long will it take?
- What additional evidence do I need?
- Should I file or settle?

### Step 7: File Your Case

#### 7.1 Final Checklist
- [ ] Evidence collected and verified
- [ ] Lawyer engaged
- [ ] Application drafted (using templates)
- [ ] Court fees calculated
- [ ] Jurisdiction confirmed
- [ ] Within limitation period
- [ ] Compensation accepted "under protest"

#### 7.2 Filing Process
1. Submit application to Collector/Court
2. Pay court fees
3. Obtain acknowledgment
4. Keep copies of everything
5. Note down case number

## üí° Tips & Best Practices

### For Claimants

#### ‚úÖ DO:
1. **Act Quickly**
   - File within 6 weeks/months of award
   - Don't wait for other claimants

2. **Accept Under Protest**
   - Always write "under protest" when accepting compensation
   - Or file reference immediately

3. **Collect Strong Evidence**
   - Get minimum 3 comparable sale deeds
   - Find comparable judgments (same locality)
   - Obtain Collector's Guidelines

4. **Engage Qualified Lawyer**
   - Specialized in land acquisition
   - Experienced in your High Court
   - Good track record

5. **Be Patient**
   - Cases take 3-8 years typically
   - Attend all hearings
   - Keep all documents safe

#### ‚ùå DON'T:
1. **Don't Delay**
   - Beyond 6 months needs condonation
   - Beyond 2 years very difficult

2. **Don't Fence-Sit**
   - Don't wait for others' outcomes
   - File your own case independently

3. **Don't File Without Evidence**
   - No evidence = waste of money
   - Need minimum comparable evidence

4. **Don't Accept Without Protest**
   - Bars your right to file reference
   - Always protest in writing

5. **Don't Ignore Procedures**
   - Follow court rules strictly
   - Pay correct court fees
   - File in correct jurisdiction

### For Lawyers

#### ‚úÖ DO:
1. **Assess Evidence First**
   - Check quality of comparables
   - Verify dates and locations
   - Ensure authenticity

2. **Use Templates as Base**
   - Customize for specific facts
   - Add relevant case law
   - Update with recent judgments

3. **File in Correct Court**
   - Verify jurisdiction carefully
   - Check bench coverage
   - Confirm e-filing availability

4. **Set Realistic Expectations**
   - Explain timeline (years)
   - Discuss costs upfront
   - Show success probability

5. **Keep Client Informed**
   - Regular updates
   - Explain legal terms
   - Share court orders promptly

#### ‚ùå DON'T:
1. **Don't Guarantee Success**
   - Depends on evidence and facts
   - Court's discretion involved

2. **Don't Miss Deadlines**
   - Limitation periods critical
   - Response deadlines important

3. **Don't Ignore Precedents**
   - Study similar cases
   - Follow successful strategies
   - Cite relevant judgments

## üîß Troubleshooting

### Application Issues

#### App Won't Start
**Problem**: Application doesn't open  
**Solution**:
```bash
# Check if Node.js is installed
node --version

# If not installed, download from nodejs.org
# Then try again:
cd landacquisition
npm install
npm start
```

#### Port Already in Use
**Problem**: Error "Port 3000 already in use"  
**Solution**:
```bash
# Windows
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# Or change port in package.json
```

#### Blank Page
**Problem**: Application opens but shows blank page  
**Solution**:
1. Clear browser cache (Ctrl+Shift+Delete)
2. Try different browser
3. Check browser console (F12) for errors

### Search Not Working
**Problem**: Search returns no results  
**Solution**:
1. Check spelling
2. Try simpler terms
3. Use single words instead of phrases
4. Try different categories

### Templates Not Copying
**Problem**: Copy button doesn't work  
**Solution**:
1. Manually select text (Ctrl+A)
2. Copy (Ctrl+C)
3. Paste in your document (Ctrl+V)

### Links Not Opening
**Problem**: External links don't open  
**Solution**:
1. Right-click link
2. Select "Open in new tab"
3. Check if popup blocker is enabled
