# PDF Drafts Folder

This folder contains all 50 legal draft PDFs.

## Structure

- **Reference Applications**: 20 PDFs (Draft_1 to Draft_20)
- **Appeals**: 20 PDFs (Draft_21 to Draft_40)
- **Reviews**: 10 PDFs (Draft_41 to Draft_50)

## File Naming Convention

```
Draft_[ID]_[Title].pdf
```

Example:
- `Draft_1_Delay_Condonation_with_Financial_Hardship.pdf`
- `Draft_21_Reference_Court_Rejection.pdf`
- `Draft_41_Error_Apparent_on_Face_of_Record.pdf`

## Access

PDFs can be accessed via:
1. **Direct URL**: `/drafts-pdf/Draft_[ID]_[Title].pdf`
2. **App Interface**: Navigate to `/pdf-drafts` page
3. **Download**: Use download button in app

## Generation

PDFs are generated from the legal drafts data and saved in this folder for easy access and distribution.

## Notes

- All PDFs are generated from template data
- Customize with specific case details before use
- Consult qualified lawyer before filing
- Ensure all required documents are attached
